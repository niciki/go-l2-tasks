# 1. Что выведет программа? Объяснить вывод программы.
```go
package main
 
import (
    "fmt"
)
 
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```
Программа выведет: ```[77, 78, 79]``` в слайс `b` положится слайс массива `а` с 1ого по 4ый индекс не включительно.
